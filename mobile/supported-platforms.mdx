QorellyMobile is built with Flutter, which compiles to native code for each platform. This means the app runs with full native performance on every device, not inside a web wrapper.

## Mobile — iOS

| | Details |
|---|---------|
| **Devices** | iPhone (all modern models) |
| **OS requirement** | iOS 14.0 or later |
| **Distribution** | App Store |
| **Features** | Full feature set including camera barcode scanning, Bluetooth printing, push notifications |

## Mobile — Android

| | Details |
|---|---------|
| **Devices** | Android phones from all major manufacturers |
| **OS requirement** | Android 6.0 (API 23) or later |
| **Distribution** | Google Play Store |
| **Features** | Full feature set including camera barcode scanning, Bluetooth printing, push notifications, native POS hardware integration |

<Info>
  Android devices with built-in POS hardware (such as Sunyard terminals) get native printer integration — no Bluetooth pairing needed.
</Info>

## Tablets — iPad

| | Details |
|---|---------|
| **Devices** | iPad, iPad Air, iPad Pro, iPad Mini |
| **OS requirement** | iPadOS 14.0 or later |
| **Distribution** | App Store (same app as iPhone, optimized for iPad) |
| **Layout** | Multi-column grids, split views, enhanced floor plan |

iPads are ideal for:

- **Table-side ordering** — Waiters carry an iPad to take orders at the table
- **Front desk / reception** — Spas and salons use iPad as a check-in station
- **Kitchen display** — Mount an iPad in the kitchen to show incoming orders
- **Floor plan management** — The larger screen makes the interactive floor plan easy to use

## Choosing the right device for your business

<AccordionGroup>
  <Accordion title="Restaurant counter / cashier" icon="cash-register">
    **Recommended:** Android tablet or iPad

    A tablet mounted at the counter gives cashiers a large touchscreen for the POS. Use **POS mode** or **Quick Service mode** for a focused interface.
  </Accordion>

  <Accordion title="Table-side ordering" icon="utensils">
    **Recommended:** iPad Mini or Android phone

    Compact and portable for waiters to carry between tables. Use **Full Service mode** for access to the floor plan and table assignment.
  </Accordion>

  <Accordion title="Spa / salon reception" icon="spa">
    **Recommended:** iPad or Android tablet

    Place at the front desk for appointment check-in and checkout. Use **Bookings mode** for a scheduling-focused interface.
  </Accordion>

  <Accordion title="Back office / manager" icon="desktop">
    **Recommended:** Desktop (macOS or Windows)

    For managers reviewing reports, managing inventory, and configuring settings. The desktop layout provides the most information density.
  </Accordion>

  <Accordion title="Delivery / on-the-go" icon="motorcycle">
    **Recommended:** iPhone or Android phone

    For delivery staff or business owners checking in on the go. Offline mode ensures orders are never lost even with spotty connectivity.
  </Accordion>
</AccordionGroup>
