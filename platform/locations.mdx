If your business operates from more than one location, Qorelly's multi-location feature lets you manage all branches from a single account.

## Setting up locations

Navigate to **Settings > Locations** to manage your branches.

### Adding a location

1. Click **Add Location**
2. Fill in the location details:

| Field | Description |
|-------|-------------|
| Name | Branch name (e.g., "Victoria Island Branch") |
| Address | Physical address |
| City | City |
| State | State or region |
| Phone | Branch contact number |
| Email | Branch email address |
| Operating hours | Business hours for this location |

3. Save the location

### Default location

One location is set as the default. New orders, bookings, and data are associated with the active location unless changed.

## Switching locations

Use the location selector in the dashboard top bar to switch between branches. When you switch:

- All dashboard metrics update to show data for the selected location
- POS, bookings, and inventory are filtered to that location
- New orders and bookings are created under the selected location

## Location-specific data

Each location has its own:

- Orders and revenue
- Bookings and reservations
- Inventory stock levels
- Team member assignments

## Enabling and disabling locations

You can temporarily disable a location without deleting it. Disabled locations:

- Do not appear in the location selector
- Retain all historical data
- Can be re-enabled at any time

For more details on how multi-location works across the platform, see [Multi-Location Management](/administration/multi-location).
