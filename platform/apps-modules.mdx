Qorelly is modular — you choose which features to enable based on your business needs.

## Available modules

Navigate to **Settings > Apps** to toggle modules on or off.

| Module | Description |
|--------|-------------|
| Menu | Digital menu management with categories and items |
| POS | Point-of-sale system for creating and processing orders |
| Inventory | Stock tracking, suppliers, and purchase orders |
| Bookings | Reservations, table management, and floor plan |

<Note>
  At least one module must be enabled at all times.
</Note>

## How modules affect the platform

### When a module is enabled

- Its section appears in the sidebar navigation
- Its items appear on the Dashboard Apps tab
- Related features are accessible throughout the platform

### When a module is disabled

- Its section is hidden from the sidebar
- Its data is preserved but not accessible
- Related features (e.g., low stock alerts in POS) are hidden

## Default modules by business type

Different [business types](/business-types) come with different default modules:

| Business Type | Default Modules |
|---------------|----------------|
| Restaurant | Menu, POS, Bookings |
| Spa / Salon | Menu, Bookings |
| Pharmacy | POS, Inventory |
| Retail | POS, Inventory |
| Hospitality | Menu, Bookings |
| Fuel Station | POS, Inventory |

You can always customize which modules are enabled regardless of your business type.

## Module interactions

Some modules work together:

- **Menu + POS** — Menu items appear in the POS for order creation
- **Menu + Inventory** — Menu items can have inventory tracking enabled
- **POS + Inventory** — Orders automatically update stock levels
- **Bookings + POS** — Bookings can be linked to orders for payment processing

<Tip>
  Start with the modules you need today. You can enable additional modules at any time without losing any data.
</Tip>
